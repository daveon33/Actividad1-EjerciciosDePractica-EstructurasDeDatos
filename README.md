# Ejercicio: Arreglos y Promedio

## Objetivo

El objetivo de este ejercicio es entender cómo funciona el arreglo como estructura de datos para almacenar información de manera ordenada.

## Descripción

El programa utiliza tres arreglos:

- **Arreglo 1 y 2:** Sus valores son ingresados por el usuario.
- **Arreglo 3:** Almacena la diferencia entre cada par de valores correspondientes del primer y segundo arreglo.

Una vez llenado el tercer arreglo, el programa calcula el promedio de sus elementos: suma todos los valores y divide el resultado entre el número de elementos, mostrando finalmente el promedio en pantalla.
